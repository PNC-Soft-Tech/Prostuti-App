## 🖼️ Detailed Design Elements

### **1. Typography**
```css
Main Title (প্রস্তুতি): 
- Font: Noto Sans Bengali (Bold)
- Size: 60px
- Color: #FFFFFF
- Position: Left side, top section

Subtitle (Your Career Preparation Companion):
- Font: Inter (Medium)
- Size: 24px
- Color: #FFFFFF
- Opacity: 90%

Feature List:
- Font: Inter (Regular)
- Size: 18px
- Color: #FFFFFF
- Line height: 1.5
```

### **2. Color Palette**
```css
Primary: #50BDB4 (Teal - from app theme)
Secondary: #2E8B80 (Darker teal)
Accent: #FFFFFF (White text)
Success: #4CAF50 (Green for checkmarks)
Background Gradient: linear-gradient(135deg, #50BDB4 0%, #2E8B80 100%)
```

## 📋 Step-by-Step Creation Guide

### **Step 1: Setup**
1. Create new document: 1024×500px
2. Set background gradient: #50BDB4 to #2E8B80
3. Add guides at 400px from left (text/visual split)

### **Step 2: Text Section (Left)**
1. Add app logo at top-left
2. Place "প্রস্তুতি" title below logo
3. Add English subtitle
4. List key features with checkmarks
5. Add call-to-action button

### **Step 3: Visual Section (Right)**
1. Place phone mockup (centered)
2. Add app screenshot inside mockup
3. Scatter educational icons around
4. Add geometric shapes for depth

### **Step 4: Final Touches**
1. Ensure text is readable
2. Check contrast ratios
3. Add subtle shadows/gradients
4. Export as PNG (high quality)
